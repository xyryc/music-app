# Upgrading Expo

## Purpose
Helps upgrade Expo SDK and dependencies safely.

## Capabilities
- Check current Expo version and compatibility
- Plan upgrade paths between SDK versions
- Update package.json dependencies
- Handle breaking changes and migrations
- Update native configuration files
- Test upgraded app functionality

## When to Use
- New Expo SDK version available
- Dependencies are outdated
- Need features from newer SDK
- Preparing for app store submission

## Guidelines
- Backup before upgrading
- Review official changelog
- Update in small increments
- Test on both iOS and Android
- Update expo-doctor after upgrade
