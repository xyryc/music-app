# API Integration Helper

## Purpose
Assists with integrating APIs and handling data in Expo/React Native apps.

## Capabilities
- Create API service layers and clients
- Implement authentication flows
- Handle request/response typing
- Manage loading and error states
- Set up data caching strategies
- Configure environment variables

## When to Use
- Connecting to REST or GraphQL APIs
- Implementing authentication
- Syncing data with backend
- Handling offline scenarios

## Guidelines
- Use TypeScript for API types
- Implement proper error handling
- Add request/response interceptors
- Secure sensitive data
- Follow API response conventions
