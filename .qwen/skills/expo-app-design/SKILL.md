# Expo App Design

## Purpose
Guides visual design, theming, and UX decisions for Expo apps.

## Capabilities
- Implement design systems and themes
- Set up dark/light mode support
- Create consistent spacing and typography
- Design navigation patterns
- Ensure accessibility compliance
- Review UI/UX best practices

## When to Use
- Starting a new app design
- Implementing a design system
- Adding theme support
- Improving accessibility
- Redesigning existing screens

## Guidelines
- Follow platform conventions (iOS/Android)
- Maintain consistent spacing scale
- Use semantic colors
- Ensure sufficient contrast ratios
- Support dynamic type sizing
