# Expo Feature Builder

## Purpose
Helps build new features for Expo/React Native applications from concept to implementation.

## Capabilities
- Scaffold new feature directories with proper structure
- Create feature components, screens, and navigation
- Implement business logic and state management
- Add feature-specific hooks and utilities
- Write feature tests and documentation

## When to Use
- User requests adding a new feature to the app
- Need to scaffold feature boilerplate
- Implementing user stories or requirements

## Guidelines
- Follow existing project conventions
- Use TypeScript for type safety
- Keep components small and focused
- Add appropriate error handling
