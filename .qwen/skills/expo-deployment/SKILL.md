# Expo Deployment

## Purpose
Manages building and deploying Expo apps to stores and distribution channels.

## Capabilities
- Configure EAS Build and Submit
- Set up build profiles (development, preview, production)
- Manage app credentials and signing
- Deploy to TestFlight and Google Play Internal Testing
- Configure OTA updates with EAS Update
- Handle environment-specific configurations

## When to Use
- Preparing app store release
- Setting up CI/CD pipelines
- Distributing builds to testers
- Pushing over-the-air updates

## Guidelines
- Use EAS Build for production builds
- Configure proper build profiles
- Set up automatic submissions
- Test OTA updates thoroughly
- Maintain version numbering
